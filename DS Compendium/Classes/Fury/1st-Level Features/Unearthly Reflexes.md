# Unearthly Reflexes

*Elusive as a hummingbird.*

|                    |                     |
| :----------------- | :------------------ |
| **Keywords:** --   | **Type:** Triggered |

**Spend 1 Rage**: You reduce the potency of any effect associated with the damage for you by 1.
