# Blot Out The Sun! (11 Focus)

*What makes a good soldier? The ability to fire four shots a minute in any weather.*

|                                      |                                 |
| :----------------------------------- | :------------------------------ |
| **Keywords:** Ranged, Strike, Weapon | **Type:** Action                |
