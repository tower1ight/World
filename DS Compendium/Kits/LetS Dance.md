# Let's Dance

*Keeping your enemies stumbling around the battlefield is second nature to you.*

|                                     |                        |
| :---------------------------------- | :--------------------- |
| **Keywords:** Melee, Strike, Weapon | **Type:** Action       |
| **Distance:** Melee 1               | **Target:** 1 creature |
